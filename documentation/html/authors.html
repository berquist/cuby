Authors and Contact

Cuby is developed by Jan Řezáč at the Institute of Organic Chemistry and Biochemistry AS CR.

Parts of the framework were written by:

  • Tomáš Kubař - core of the MD engine
  • Jakub Klemsa - the PLBFGS optimizer and some other optimization-related code

An important contribution is also the testing of the development versions by many of my coworkers and collaborators.

How to cite Cuby

Any work based on calculations performed using Cuby should cite both the software and the paper where it was introduced:

  • J. Řezáč, Cuby – ruby framework for computational chemistry, version 4, http://cuby4.molecular.cz
  • J. Řezáč, J. Comput. Chem. 2016, DOI: 10.1002/jcc.24312
